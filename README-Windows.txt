LATTE LAB MENU - WINDOWS 免安装版
================================

无需安装 Python 或其他软件。

启动：
1. 完整解压 ZIP 文件。
2. 双击 "Start Latte Lab Menu.bat"。
3. 菜单会自动在浏览器中打开。
4. Windows 防火墙首次询问时，请允许“专用网络”访问。

手机访问：
1. 手机与 Windows 电脑连接同一个 WiFi。
2. 扫描菜单右上角的二维码。
3. 每次启动软件时，二维码会根据当前 WiFi 地址重新生成。

更新菜单：
- 编辑 "Food panda LAB.xlsx"。
- 在 "image" 文件夹加入与英文饮品名称同名的图片。
- 将 Promotion 设置为 1，饮品会自动加入顶部推荐轮播。
- 软件运行时会自动读取 Excel 和新增图片。

停止：
- 双击 "Stop Latte Lab Menu.bat"。

注意：
- 请保持所有文件和文件夹在一起。
- 不要单独移动启动文件或 runtime 文件夹。
- 支持 64 位 Windows 10 和 Windows 11。
